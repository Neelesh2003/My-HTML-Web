HTML : Hyper Text Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to build and structure the content of webpages. Think of it as the skeleton of any website. Once you understand HTML, you can start adding style with CSS and interactivity with JavaScript. In this document, we’ll walk through the most essential HTML elements with clear explanations and examples.

What is HyperText and Markup Language?

  • HyperText: This means you can connect one webpage to another using hyperlinks. For example, clicking on a link in an article that takes you to a reference page.
  • Markup Language: A markup language uses special tags to “mark up” text, giving meaning and structure. For example, wrapping text in <h1> tells the browser it is a main heading, not just plain text.

1. Headings (<h1> to <h6>)

Headings are like titles in a book. They define sections and subsections in your content. Search engines and screen readers rely on headings to understand the structure of your page. The size of the text decreases from <h1> to <h6>, but the main purpose is structural importance, not style. 

<h1>Main Title</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>

Definition: Headings establish a hierarchy. <h1> is the top-level title, <h2> is for big sections, <h3> is for smaller sections within, and so on. You should normally use only one <h1> per page.

2. Paragraph (<p>)

A paragraph is a block of text used for regular content. Browsers automatically add spacing before and after each paragraph so that they are visually separated.

<p>This is the first paragraph.</p>
<p>This is another paragraph.</p>

Definition: Paragraphs are meant for readable chunks of text. Instead of pressing Enter multiple times, you should always use a new <p> for each idea.

3. Horizontal Rule (<hr>)

The <hr> tag represents a thematic break. It’s often displayed as a horizontal line. It separates ideas or sections.

<p>Introduction</p>
<hr />
<p>Next Topic</p>

Definition: Think of it like drawing a line in your notebook when you want to move to a new idea.

4. Line Break (<br>)

The line break tag tells the browser to stop writing on the current line and continue on the next line. Unlike <p>, it does not create extra spacing, it just moves the text down one line. 

<p>Address Example:<br>
221B Baker Street<br>
London, England</p>

Definition: Use <br> for small breaks (like in addresses or poems), not for creating paragraphs.

5. Lists (<ul>, <ol>, <li>)

Lists are used to group items. They make information easier to read and scan. There are two main types:

  • <ul> — unordered list (bullets)
  • <ol> — ordered list (numbers)
  • <li> — list item (goes inside <ul> or <ol>)
<ul>
  <li>Tea Bag</li>
  <li>Water</li>
  <li>Sugar</li>
  <li>Milk</li>
</ul>

<ol>
  <li>Boil water</li>
  <li>Add tea, sugar, milk</li>
  <li>Boil 5 minutes</li>
  <li>Serve</li>
</ol>

Definition: Use lists whenever you have multiple items to present. <ul> is for items without a sequence (like groceries). <ol> is for steps in order (like recipes).

6. Links & Hyperlinks (<a>)

Hyperlinks are what make the web connected. The <a> tag creates a clickable link. It requires the href attribute to specify the destination.

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

Definition: The <a> tag stands for “anchor.” It can link to external sites, other pages on your site, or even different sections of the same page.

target attribute

By default, links open in the same tab. Use target="_blank" to open in a new tab or window.

<a href="https://example.com" target="_blank">Open Example</a>

Internal links

You can link within the same page using IDs. This is useful for navigation menus or “Back to top” buttons. 

<a href="#summary">Go to Summary</a>

7. Images (<img>)

The <img> tag inserts an image into your page. It is self-closing and needs two key attributes:

  • src — the file path or URL of the image
  • alt — alternative text that describes the image (important for accessibility and SEO)
<img src="/HTML-Web/Images/Image-01.png" alt="Beagle dog" height="200" />

Definition: Images add visual content to your webpage. The alt attribute ensures that users with screen readers or poor internet connections can still understand what the image represents.

Quick Summary

Here’s a recap of what we covered:

  • <h1> - <h6>: Headings, used to structure your document.
  • <p>: Paragraphs, used for text blocks.
  • <hr>: Horizontal line to separate sections.
  • <br>: Line break, moves text to the next line.
  • <ul>/<ol>/<li>: Lists, unordered (bullets) or ordered (numbers).
  • <a>: Links, create navigation to other pages or sections.
  • <img>: Images, used to add visuals to your content.
